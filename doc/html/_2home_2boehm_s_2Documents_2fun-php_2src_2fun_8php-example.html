Takes a predicate and a iterable and returns an array containing the members of the given iterable which satisfy the given predicate.

((a, i, [a]) → Bool) → [a] → [a]
Parameters
callable$predicate
iterable$arr
Returns
array

$isEven = function($n) { return $n % 2 == 0; }; F::filter($isEven, [1, 2, 3, 4, 5]); //=> [2, 4]
